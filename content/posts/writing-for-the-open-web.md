# Writing for the Open Web

Good blog infrastructure should disappear behind the writing.

That means pages should load quickly, headings should be meaningful, and feeds should exist without extra ceremony.

## Durable by default

Static content is still a strong baseline. It keeps complexity low and lets the page itself do the work.

## Elegant systems leave room for words

A blog template succeeds when it makes publication feel simple instead of precious.
